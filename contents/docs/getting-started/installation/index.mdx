## Overview

Welcome to the LogosQ installation guide. Here, you'll find all the information you need to get started.

## Prerequisites

Before proceeding, ensure you have the following:

- Basic knowledge of command-line interface (CLI)
- A code editor (e.g., VSCode)

## Installation Steps

Since this is a library in Rust, one can directly add the dependencies and use the necessary functions. No furthur needs to worry about the the installation! In some cases, Rust on MacOS and Windows may have implementation types different from Ubuntu, such scenarios need further adaption and installation of the package on Software system.
